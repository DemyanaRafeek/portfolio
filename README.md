# Demyana Rafeek | Flutter Developer Portfolio

A modern, dark-themed portfolio website showcasing skills, projects, experience, and contact information.

## Features

- **Hero Section** - Introduction with profile and CTA buttons
- **About Me** - Professional summary with highlighted achievements
- **Education & Certificates** - Academic background and certifications
- **My Skills** - Technical, soft skills, tools, and languages
- **Experience** - Timeline with Challenge/Action/Result format
- **My Services** - App Development, Performance Tuning, Backend Binding
- **Featured Projects** - PowerGym, Shozella, Toku App
- **Achievements** - DEPI Selection, 3+ Projects, 30% Performance Boost
- **Testimonials** - Client and instructor feedback
- **Contact** - Form and social links
- **Footer** - Quick links and social media

## Customization

1. **Profile Image**: Replace the `.profile-placeholder` in the hero with an `<img>` tag pointing to your photo
2. **CV Download**: Update the `href` on the "Download CV" button to your CV file
3. **GitHub Links**: Add your project repository URLs to the "View on GitHub" buttons
4. **Social Links**: Update the `href` attributes for GitHub, LinkedIn, and WhatsApp
5. **Contact Form**: Integrate with Formspree, EmailJS, or your backend API

## Running Locally

Simply open `index.html` in a browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

Then visit `http://localhost:8000`
